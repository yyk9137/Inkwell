# Inkwell

Lightweight SillyTavern-like chat interface for LLM character interactions.

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Express + TypeScript
- **Storage**: JSON files (migrate to SQLite later)
- **Styling**: Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```
inkwell/
├── client/          # React frontend
├── server/          # Express backend
├── shared/          # Shared types
└── config/          # Configuration
```
